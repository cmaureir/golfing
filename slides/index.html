General tips

Multiple conditions

        
        if i == 42 and j == "dog" and k == [1,2,3]:
            pass
        
    
        
        if (i,j,k)==(42,"dog",[1,2,3]):
            pass

if-else

        
        if a<b:
            return c
        else:
            return d
        
    
        
        return(d,c)[a<b]

if and booleans

        
        if a and b:
            pass
        
    
        
        if a*b:
            pass

nested dependant for

        
        for i in range(m):
            for j in range(n):
                do_stuff(i,j)
        
    
        
        for k in range(m*n):
          do_stuff(k//n,k%n)

    Pyth

    (no, I did not misspell it)

    Pyth

    • An official golfing language
    • JIT compiled language
    • pyth.readthedocs.io
    • Uses prefix notation: 2+2 is +2 2
    • Operators (+, -, *, /, %)
    • Negative numbers: -4 needs to be _4
    • Pre defined variables: Z=0, T=10, k="", d=" ", b="\n"

    For loop

            
        FNrZTN
    • F: for loop
    • N: iterator
    • r: range()
    • Z: 0
    • T: range limit
    • N: variable to print
